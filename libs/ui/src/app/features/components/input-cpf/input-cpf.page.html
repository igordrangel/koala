Input CPF The Input CPF component is a specialized input field designed for entering Brazilian CPF (Cadastro de Pessoas Físicas) numbers. It provides built-in validation to ensure that the entered CPF number is in the correct format and is valid according to the CPF rules. This component enhances user experience by guiding users to input their CPF correctly and providing immediate feedback on any errors. Installation Use the Koala CLI to generate a new Input CPF component. Usage
