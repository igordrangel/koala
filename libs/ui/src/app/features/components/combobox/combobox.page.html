Combobox A searchable select built with Angular Aria, supporting local or remote filtering through a resource and dedicated loading and empty states. Local Filtering When you pass a static option list, the component filters it internally and behaves like a searchable select. You can also enable multiple selection.

Selected value

{{ localComboboxControl.value || 'No item selected' }}

Multiple values 

{{
                localMultipleComboboxControl.value.length
                  ? localMultipleComboboxControl.value.join(', ')
                  : 'No items selected'
              }}

Filtering happens inside the component by comparing the typed text against each option label.

Remote Filtering With Resource The component accepts a factory that receives the filter signal and creates a resource, httpResource, or rxResource internally to fetch matching results.

Selected value

{{ remoteComboboxControl.value || 'No item selected' }}

Typing updates the filter signal and triggers the resource. While results are loading, the field shows a spinner and the dropdown replaces the list with a Searching... message.
