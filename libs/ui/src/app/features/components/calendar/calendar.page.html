Calendar Calendars can be used to select dates in a form. Installation Use the Koala CLI to generate a new calendar component. Date picker The calendar component can also be used as a date picker. To do this, you can use the popover directive to display the calendar when the user clicks on an input field. The input-calendar demo below includes the date, datetime, month and daterange modes.
date value: {{ inputDateValue() }}
datetime value: {{ inputDateTimeValue() }}
month value: {{ inputMonthValue() }}
daterange value: {{ inputDateRangeValue() }}
Calendar The calendar component is a date picker that can be used to select dates in a form. It is built using the Cally library, which provides a simple and customizable API for creating date pickers.
