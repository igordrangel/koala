Input CNPJ The Input CNPJ component is a specialized input field designed for entering Brazilian CNPJ (Cadastro Nacional da Pessoa Jurídica) numbers. It provides built-in validation to ensure that the entered CNPJ number is in the correct format and is valid according to the CNPJ rules. This component enhances user experience by guiding users to input their CNPJ correctly and providing immediate feedback on any errors. Installation Use the Koala CLI to generate a new Input CNPJ component. Usage
