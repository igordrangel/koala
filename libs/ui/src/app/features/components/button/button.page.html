Button A simple button component. Installation Use the Koala CLI to generate a new button component. Variants The button component supports several variants to fit different use cases and design requirements.
Sizes The button component supports different sizes to accommodate various design needs.
Outline The outline variant provides a button style with a border and transparent background, ideal for secondary actions.
Soft The soft variant offers a button style with a softer background color, suitable for less prominent actions.
Dash The dash variant provides a button style with a dashed border, suitable for less prominent actions.
Disabled The disabled state indicates that the button is not interactive and cannot be clicked.
Circle The circle variant provides a button style with a circular shape, ideal for icon buttons.
