Modal A simple modal component. Installation Use the Koala CLI to generate a new modal component. Dialog modal The dialog modal is a classic modal component that can be used to display content in a centered overlay. It is commonly used to display important information or prompt the user for a decision.
Dialog modal, closes when clicked outside The dialog modal can be configured to close when the user clicks outside of it. This is a common behavior for modals that are used to display non-critical information or that do not require user interaction.
Dialog modal with a close button at corner The dialog modal can be configured to have a close button at the corner. This is a common design pattern for modals that are used to display critical information or that require user interaction.
Dialog modal with custom width The dialog modal can be configured to have a custom width. This is useful for modals that need to display more content or require a specific layout.
